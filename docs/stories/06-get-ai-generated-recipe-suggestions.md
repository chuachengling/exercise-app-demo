# User Story: 6 - Get AI-Generated Recipe Suggestions

**As a** user,
**I want** to receive personalized healthy recipe suggestions,
**so that** I can discover new nutritious meals that align with my health goals.

## Acceptance Criteria

* Users can access a recipes page with healthy meal suggestions
* Recipe suggestions are generated using AI based on user's health goals and preferences
* Each recipe includes ingredients list, preparation instructions, and nutritional information
* Users can save favorite recipes for future reference
* Recipe suggestions refresh periodically or on user request
* App works offline with cached recipes, but requires internet for new AI-generated content

## Notes

* Uses natural language AI (ChatGPT or local LLaMA model) for recipe generation
* Recipes should align with user's selected health goals from signup
* Consider dietary restrictions and preferences in future iterations
