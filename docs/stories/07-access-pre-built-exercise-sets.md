# User Story: 7 - Access Pre-Built Exercise Sets

**As a** user,
**I want** to access a catalog of pre-built exercise sets (HIIT, weightlifting, arms-focused),
**so that** I can follow structured workout routines without having to plan them myself.

## Acceptance Criteria

* App includes a catalog of pre-built exercise sets for different workout types
* Exercise sets are categorized by type (HIIT, weightlifting, arms-focused, etc.)
* Each exercise set includes detailed instructions, duration, and difficulty level
* Users can start an exercise set and follow along with the routine
* Completed exercise sets are automatically logged to the user's exercise history
* Exercise sets are stored locally and don't require internet access

## Notes

* This provides immediate value to users who want guided workouts
* Should complement the custom exercise generation feature (separate story)
* Exercise sets should align with different user goals established during signup
