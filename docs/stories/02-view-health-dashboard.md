# User Story: 2 - View Health Dashboard

**As a** user,
**I want** to see a dashboard on my homepage that displays my tracked health data over a customizable time period,
**so that** I can quickly understand my progress and patterns.

## Acceptance Criteria

* Homepage displays a dashboard with visual summaries of recent activity
* Users can select the time period for data display (days, weeks, months)
* Dashboard shows key metrics: exercise sessions, food intake, water consumption
* Visual representations (charts/graphs) make data easy to understand at a glance
* Dashboard updates automatically when new data is tracked

## Notes

* This serves as the central hub for users to monitor their health journey
* Time period selection allows users to focus on recent activity or longer-term trends
